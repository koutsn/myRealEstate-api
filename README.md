# Real Estate api

## Endpoints:
    /login
        Login to the system
    /register
        Create a new user 
    /user/{username}
         Get user information by username
    /users
        Get list of all users
    /objects
        Get list of all real estate objects
    /object/{id}
        Get real estate object by ID
    /object/id/images
        Upload images for a real estate object by ID
